Introduction¶

We will investigate TMDb movie data (source: Kaggle). This dataset contains information about 5,000 movies collected from The Movie Database (TMDb), including user ratings and revenue.

We want to answer these questions by exploring that dataset:

  • Question 1: Which genres are most popular from year to year?
  • Question 2: How distribution of movie budget and movie revenue?
  • Question 3: What kinds of properties are associated with movies that have high revenues?

First, set up import statements for all of the packages that we will use.

Conclusions¶

  • Originally, we get 4803 observations with 20 columns from TMDb movie data (source: Kaggle).
  • We didn't found any duplicate data in the dataset.
  • We did data cleaning which resulted only 3734 observations that will be used in exploration data analysis.
    • Fix wrong data types for release_date (str to datetime), revenue (int to float), and budget (int to float) columns.
    • Remove 1069 observations due to missing release_date data and invalid budget data (budget less than $1000 USD).
    • Replace runtime value with None for 2 observations which have runtime equal to 0.
    • Standardize multiple values format by using ';' separator for genres, keywords, production_companies, production_countries, and spoken_language columns.
    • Create separate genre dataframe which contain data which is the result of normalization from column genres to be able to answer research question: Which genres are most popular from year to year?
  • We used cleaned dataset which contains 3734 observations in exploratory data analysis. Here are insights that we get from answeing research questions:
    • Based on 30-year moving average popularity in 1990-2016, Fantasy, Adventure, Science Fiction, Action, and Thiller are the most popular genres. Those genres have 30-year moving average popularity higher than the median of 30-year moving average popularity.
    • From statistic summary of moview budget and revenue, 50% of movies have budget \$23.3M USD and get revenue \$39.4M USD.
    • Number of votes, budget amount, and movie popularity score are properties which have moderate positive correlation with revenue.
